## Getting Started

**1- Clone this repository:**
```bash
git clone https://github.com/luizfranzon/CRUD.git
```
**2- Install the dependencies:**
```bash
npm install or yarn
```
**3- Run the application:**
```bash
npm run dev or yarn dev
```
